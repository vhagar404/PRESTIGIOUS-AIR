Prestigious Air is a modern flight booking web application that allows users to search flights between cities worldwide, select travellers, add extras, and complete a booking.

## Features

- Search flights by **country and city**
- Dynamic flight pricing based on **distance**
- Traveller selection
- Class selection (Economy, Business, First)
- Passenger details form
- Add-ons such as:
  - Extra baggage
  - Meal
  - Lounge access
  - Travel insurance
- Payment method selection (Card / PayPal)
- Booking confirmation with reference number
- Mobile responsive design

## Built With

- React
- Vite
- Tailwind CSS
- JavaScript
- React Icons

## Installation

Clone the repository:

```
git clone https://github.com/vhagar404/PRESTIGIOUS-AIR.git
```

Go into the project folder:

```
cd prestigious-air
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```
